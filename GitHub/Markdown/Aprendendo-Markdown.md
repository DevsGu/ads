# 📖 Aprendendo Markdown

Para aprender os conceitos de **Markdown**, utilizei uma técnica de
**memorização** com **revisão espaçada**. Basicamente, criei um pequeno
**storytelling** para associar cada recurso da linguagem a uma imagem
mental, facilitando a lembrança da sintaxe. 🧠✨

------------------------------------------------------------------------

## 📝 Exemplo

Criei títulos do **maior (`#`) ao menor (`##`)**.

O **Grande Homem**, escrito em **negrito**, foi
<ins>sublinhado</ins> e depois ~~tachado~~.

``` text
Coitado... Mal sabia ele o que estava escrito em seu código.
```
