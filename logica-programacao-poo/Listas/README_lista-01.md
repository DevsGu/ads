## 📝 Exercícios

Esta lista contém **31 exercícios**, compostos por questões conceituais
e desafios práticos de programação em Java.

Os exercícios envolvem:

-   Perguntas teóricas;
-   Desenvolvimento de algoritmos;
-   Implementação de métodos;
-   Criação de classes;
-   Aplicação dos conceitos de Programação Orientada a Objetos;
-   Resolução de problemas utilizando lógica de programação.

------------------------------------------------------------------------

## 🎯 Objetivo

-   Consolidar os fundamentos da linguagem Java;
-   Desenvolver o raciocínio lógico;
-   Praticar estruturas de controle;
-   Aplicar conceitos iniciais de Programação Orientada a Objetos;
-   Resolver problemas por meio da implementação de algoritmos.
