# 💻 Lógica de Programação e Programação Orientada a Objetos

Repositório destinado aos meus estudos de Lógica de Programação e Programação Orientada a Objetos (POO), reunindo listas de exercícios, desafios e projetos desenvolvidos ao longo da graduação em Análise e Desenvolvimento de Sistemas (ADS).

O objetivo deste repositório é consolidar a base da programação por meio da prática, evoluindo desde algoritmos e estruturas de controle até os principais conceitos da Orientação a Objetos utilizando Java.
