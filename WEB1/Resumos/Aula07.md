# Atividade Teórico-Prática 05 - Manutenção de Estado com Cookies e Sessões

## 📚 Sobre a Atividade

Esta atividade tem como objetivo aplicar os conceitos de **manutenção de estado em aplicações Web Java**, utilizando **Servlets**, **JSPs** e os fundamentos do protocolo HTTP.

Durante o desenvolvimento foram abordados temas essenciais para aplicações web modernas, como gerenciamento de sessões, utilização de cookies, autenticação de usuários e boas práticas de segurança em aplicações Jakarta EE.

## 🎯 Objetivos

* Compreender o funcionamento de cookies e sessões HTTP.
* Analisar o comportamento do cookie `JSESSIONID` em aplicações Java Web.
* Implementar mecanismos seguros de autenticação e logout utilizando Servlets.
* Desenvolver funcionalidades de persistência de preferências do usuário por meio de cookies.
* Aplicar conceitos de segurança relacionados aos atributos `HttpOnly`, `Secure` e gerenciamento adequado de sessões.

## 🛠️ Conteúdos Trabalhados

### 1. Diagnóstico de Sessão HTTP

* Funcionamento do cookie `JSESSIONID`.
* Utilização de `HttpSession`.
* Verificação e manutenção do estado do usuário entre requisições.
* Análise dos atributos `HttpOnly`, `Secure` e `Path`.

### 2. Autenticação e Logout Seguro

* Criação e gerenciamento de sessões.
* Armazenamento seguro de informações do usuário.
* Invalidação de sessões.
* Remoção de cookies durante o logout.

### 3. Preferências de Usuário com Cookies

* Criação de cookies persistentes.
* Leitura de cookies em requisições subsequentes.
* Persistência da preferência de idioma do usuário.

### 4. Conceitos Teóricos

* Cookies.
* Sessões HTTP.
* Protocolo HTTP Stateless.
* Segurança em aplicações Web.
* Gerenciamento de estado em aplicações distribuídas.

## 💻 Tecnologias Utilizadas

* Java
* Jakarta EE / Servlet API
* JSP
* Apache Tomcat
* HTML
* HTTP Cookies
* HTTP Sessions


## 🚀 Aprendizados

A atividade permitiu compreender como aplicações web mantêm informações de usuários entre múltiplas requisições HTTP, explorando o uso combinado de cookies e sessões. Também foram estudadas práticas de segurança fundamentais para autenticação e gerenciamento de estado em aplicações Java Web.
